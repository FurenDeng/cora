Co-ordinate Utilities (coord)¶

A module of useful functions when dealing with Spherical Polar co-ordinates.

Transforms¶

sph_to_cart(sph_arr) Convert a vector in Spherical Polar coordinates to Cartesians.
cart_to_sph(cart_arr) Convert a cartesian vector into Spherical Polars.

Vector operations¶

sph_dot(arr1, arr2) Take the scalar product in spherical polars.

Tangent Plane¶

thetaphi_plane(sph_arr) For each position, return the theta, phi unit vectors (in spherical polars).
thetaphi_plane_cart(sph_arr) For each position, return the theta, phi unit vectors (in cartesians).

Miscellaneous¶

great_circle_points(sph1, sph2, npoints) Return the intermediate points on the great circle between points sph1 and sph2.
